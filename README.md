## How to Use It

Once the workspace opens, start a board and pick the layout that matches your task. You can add notes, shapes, links, or branching elements for a mind map, then rely on the collaboration tools to work with others.

Typical workflow:
1. Start a new board or load an existing workspace.
2. Arrange ideas on the adaptive canvas.
3. Apply temporal layers when you want to separate phases or milestones.
4. Use AI co-pilot features to help structure content.
5. Sync changes across your workspace and continue editing as needed.

For imported content, use the legacy importer to move older board material into the current environment.
